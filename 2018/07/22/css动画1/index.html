2.相关属性
  1. animation-name
  2. animation-duration
  3. animation-timing-function
  4. animation-iteration-count
  5. animation-delay

  6. animation-fill-mode

    none 不会对动画等待或者完成的样式进行改变
    forwords 告诉浏览器动画结束后的样式停留在最后一帧
    backwords 动画等待的那段时间内，元素的样式将设置为动画第一帧的样式
    both backwords and forwords

  7. animation-direction

  8. animation-play-state

简写

理解animation-timing-function

1.关键字
  1. ease 默认。动画以低速开始，然后加快，在结束前变慢。
  2. ease-in 先慢后快
  3. ease-out 先块后慢
  4. ease-in-out 先慢后快再慢
  5. linear 匀速

ease和ease-in-out相比，ease在结束的时候有一个更剧烈的减速

2.贝塞尔曲线

曲线越陡峭代表速度越快，曲线越平坦代表速度越慢

每条三次贝塞尔曲线都是通过四个在0和1这个范围之间的值定义的，这四个值用于表达曲线该如何绘制。

工具

3.steps

应用

1.多个动画，一个对象

.mol { animation-name: roll-in, scale-up; animation-duration: 1s, 0.75s; animation-delay: 0s, 1s; animation-timing-function: ease-in, linear; animation-iteration-count: 1; animation-fill-mode: forwards; }
